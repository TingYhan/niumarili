# 加班时长计算系统

一个基于 Node.js + Express + SQLite 的共享加班管理系统，支持多人切换、加班记录统计、公告发布、投票、已读回执，以及每日图片/视频/一言展示。

## 主要功能

- 多用户共享加班记录管理
- 按日历查看、录入、删除加班记录
- 月度、年度统计与加班之星对比
- 公告系统
  - 普通公告与投票公告
  - 公告已读回执，支持显示谁已读
  - 投票结果显示为具名投票，能看到谁同意、谁反对
  - 公告支持上传多张图片，卡片中直接显示，点击可放大查看
- 每日内容区
  - 每日壁纸
  - 每日视频
  - 每日一言
  - 每日图片与视频默认隐藏，可手动展开
- AI 工具导航
  - 国内 / 国外网页聊天
  - 国内 / 国外 AI 编程工具

## 技术栈

- 前端：原生 HTML、CSS、JavaScript
- 后端：Node.js、Express
- 数据库：SQLite（better-sqlite3）

## 运行方式

1. 安装依赖

```bash
npm install
```

2. 启动服务

```bash
npm start
```

3. 打开浏览器访问

```text
http://localhost:3000
```

## 项目结构

- `server.js`：后端接口、数据库初始化、公告和加班数据 API
- `app.js`：前端交互逻辑、统计渲染、公告和媒体内容展示
- `index.html`：页面结构与样式
- `ai-directory-data.js`：AI 工具导航数据

## 说明

- 首次启动时会自动初始化数据库表。
- 公告图片当前采用内嵌数据方式保存，适合中小图片；单次上传图片过大时会被限制。
- 每日图片和视频默认隐藏，点击“显示每日内容”后才加载。

## 开发提示

如果你在 Windows 上安装依赖时遇到 `better-sqlite3` 编译问题，通常需要确认本机具备可用的 C++ 构建环境。
